# EasyPay

Android wallet demo app showcasing bidirectional WebView communication with a Nuxt.js e-commerce web application using [WebViewJavascriptBridge](https://github.com/RDSunhy/WebViewJavascriptBridge).

Part of the **WebBridge** demo project.
